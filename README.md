## The workshop — build an agent

Pair up (2–3) and ship one workflow you can run again:

1. **Pick one thing to automate** — repeatable, eats time, doesn't need *you*.
2. **Give it the information it needs** — use the [**agent contract**](agent-contract.md).
3. **Make it fire repeatedly** — a trigger + one tool wired. Run it once.

Build it on Claude ([claude.ai](https://claude.ai) / Claude Code). Steal a contract from the file, wire one tool, run it.

## What I look for in AI-era companies

- **A high degree of automation** — the team stays tiny because agents do the work.
- **Extreme speed** — velocity between two conversations is my single best signal.
- **Real leverage** — using AI to do far more than a team your size used to.
